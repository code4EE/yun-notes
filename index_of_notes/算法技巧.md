## <a name="快速幂">快速幂求余</a>

为什么需要快速幂求余？\
比如求解`a^b % p`, 这里存在着明显的问题,如果a和b过大，很容易就会溢出

```go
func PowerMod(a, b, c int) int {
    res := 1
    a = a%c
    for b > 0 {
        if b&1 == 1 {
            res = (res*a) % c
        }
        b = b >> 1 // 除以2
        a = (a*a) % c
    }
    return res
    }
```
