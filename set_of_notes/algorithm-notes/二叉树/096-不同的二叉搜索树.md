## 题目

#### [96. 不同的二叉搜索树](https://leetcode-cn.com/problems/unique-binary-search-trees/)

给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

 

## 分析

卡特兰数
$$
Cn+1 = 2(2n+1)/(n+2) * Cn
$$


## 解答

```go
// 卡特兰数
// C0 = 1
// C n+1 =  2(2n+1)/(n+2) * Cn
func numTrees(n int) int {
    C := 1
    for i := 0; i < n; i++ {
        C = 2 * C * (2*i+1)/(i+2)
    }
    return C
}

// 动态规划
// dp[n] = sum(dp[i]*dp[n-i]) for i in range n 
func numTrees(n int) int {
    dp := make([]int, n+1)
    dp[0] = 0
    dp[1] = 1   
    for i := 1; i <= n; i++ {
        for j := 1; j <= i; j++ {
            dp[i] += dp[i] * dp[i-j]
        }
    } 
}
```
