## 题目

#### [141. 环形链表(简单)](https://leetcode-cn.com/problems/linked-list-cycle/)

检测链表是否有环

## 分析

简单题也得多做

## 解答

```go
func hasCycle(head *ListNode) bool {
    if head == nil || head.Next == nil {
        return false
    }
    slow, fast := head, head.Next
    for slow != fast {
        if fast == nil || fast.Next == nil {
            return false
        }
        slow = slow.Next
        fast = fast.Next.Next
    }
    return true
}
```
