正序递归

## 解答

```go
 // 正序递归
func swapPairs(head *ListNode) *ListNode {
    if head == nil || head.Next == nil {
        return head
    }
    rest := head.Next.Next
    newHead := head.Next
    newHead.Next = head
    head.Next = swapPairs(rest)
    return newHead
}
```
