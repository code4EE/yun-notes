## 题目



## 分析

如果偷了前面一个`i-1`, 那么i就不能偷, 

`dp[i] = max(dp[i-1], dp[i-2]+nums[i])`

## 解答

```go
// 要么第一间偷, 要么不偷
func rob(nums []int) int {
    N := len(nums)
    if N == 0 {
        return 0
    }
    dp := make([]int, N+1)
    dp[0] = 0
    dp[1] = nums[0]
    for i := 2; i <= N; i++ {
        dp[i] = max(dp[i-1], dp[i-2]+nums[i-1]) 
    }
    return dp[N]
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```
