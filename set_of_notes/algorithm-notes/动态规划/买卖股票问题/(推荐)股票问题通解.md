// 通解
// T[i][k][0] 代表第i天结束时, 最多进行K次交易后, 手里还有0个股票的最大利润(0个股票也即是卖出)
// T[i][k][1] 代表第i天结束时, 最多进行K次交易后, 手里还有1个股票的最大利润(1个股票也即是买入)
// 状态转移方程
// T[i][k][0] = max(T[i-1][k][0], T[i-1][k][1]+prices[i])
// T[i][k][1] = max(T[i-1][k][1], T[i-1][k-1][0]-prices[i])
