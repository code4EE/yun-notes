## 题目

#### [268. 丢失的数字](https://leetcode-cn.com/problems/missing-number/)

给定一个包含 `[0, n]` 中 `n` 个数的数组 `nums` ，找出 `[0, n]` 这个范围内没有出现在数组中的那个数

## 分析

利用异或的特点

## 解答

```go
func missingNumber(nums []int) int {
    // missing先设置为n
    missing := len(nums)
    for i := 0; i < len(nums); i++ {
        missing ^= i^nums[i]
    }
    return missing
}
```
